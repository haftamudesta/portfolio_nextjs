# Haftamu Desta - Portfolio <a name="about-project"></a>

**Haftamu Desta Portfolio** is a modern, responsive personal portfolio website showcasing my expertise as a Full Stack Developer and Electrical Engineer. Built with Next.js 15, TypeScript, and TailwindCSS, this portfolio features a clean design, dark/light mode, interactive animations, and a fully functional contact form.

### Key Features <a name="key-features"></a>

- **Responsive Design** - Optimized for all screen sizes (mobile, tablet, desktop)
- **Dark/Light Mode** - Theme switcher with system preference detection
- **Contact Form** - Fully functional email integration
- **Interactive Animations** - Smooth scroll animations and hover effects using Framer Motion
- **Hero Section** - Dynamic animated introduction with particle background
- **Skills Showcase** - Visual representation of technical expertise
- **Projects Gallery** - Portfolio of completed work (coming soon)
- **Downloadable Resume** - PDF resume download option

## Getting Started <a name="getting-started"></a>

To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

- [ ] Node.js (v18 or later)
- [ ] NPM or Yarn package manager
- [ ] Git installed
- [ ] A modern web browser

### Setup

Clone this repository to your desired folder:

```bash
git clone https://github.com/haftamudesta/portfolio_nextjs.git
```

```
cd portfolio_nextjs
```

- [ ] Open the file in your code editor

```
code .
```

### Install dependencies:

- [ ] Node modules:

```
npm install
```

### Usage

To run the project, execute the following command:

```
npm run dev
```
